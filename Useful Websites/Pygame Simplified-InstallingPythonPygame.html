Install Python

    The version of Python you install is important. If you install Python 2.x, you must install Pygame 2.x. Pygame Simplified version I and version II has been developed on Python 3.x so is compatibile with Python 3.x. Pygame Simplified has been tested on Python 2.x and Pygame 2 and it works fine. All versions of Python 2.7 and above should work with Pygame Simplified. Though it is recommended to have Python 3.x with Pygame 3.

  • Windows

    As an example, we will install 3.3.2 https://www.python.org/download/releases/3.3.2/ . These instuctions can be followed for Python 3.x. or Python 2.x.

      To install Python 3.1 follow this link: https://www.python.org/download/releases/3.1/

      To install Python 3.2 follow this link: https://www.python.org/download/releases/3.2/

      Python provide installation for Python 3.2.x untill Python 3.2.5. To get the rest, just add a digit in the link with the version you want. So if you want Python 3.2.5, just add the .5. So : https://www.python.org/download/releases/3.2.5/

      To install Python 3.3.0 follow this link: https://www.python.org/download/releases/3.3.0/

      To install Python 3.3.1 follow this link: https://www.python.org/download/releases/3.3.0/

      Python provide installation for Python 3.3.x untill Python 3.3.5. To get the rest, just replace the last digit in the link with the version you want. So if you want Python 3.3.5 (Which is the lastest and recomended by Python- due to bug fixes), just replace the 0 in link above with a 5.

      Here is a link to all the Python releases https://www.python.org/downloads/

    In this example, we will install the 32 bit Python, Pygame is built for 32 bit Python. Only for windows you can find a 64 bit Pygame explained here in more depth.

    Clicking on the download at the button you should see a pop up appear asking if you would like to run Python, click run and click yes to everything.

    Once Python is installed, click on your start button. You should see Python.exe click on this, then find IDLE from your windows start button, click on IDLE and type:

    print("helloworld!")

    in the terminal, you should see something like this:

    If you see the above, this means you have installed Python correctly.

  • Linux

    • Python 2

    If your operating system is Ubuntu, you can install Python by opening a terminal window (from the desktop click on Applications > Accessories > Terminal) and entering

    sudo apt-get install python2.7

    then pressing Enter. You will need to enter the root password to install Python, so ask the person who owns the computer to type in this password if you do not know it. You also need to install the IDLE software. From the terminal, type in:

    sudo apt-get install idle

    The root password is also needed to install IDLE (ask the owner of your computer to type in this password for you).

    Python 3

    To get Python 3.3.x, and any other Python 3.x. Follow this thread, which gives a great explaination. http://askubuntu.com/questions/244544/how-do-i-install-python-3-3

  • Mac OS X

    Python 2

    If you would like to use Python 2, Mac OS X already comes installed with Python 2.7. Python 2 is only compatible with Pygame 2.

    Python 3

    If you would like to use Pygame 3, you need Python 3. Installing Pygame 3 on a Mac is quite complicated, however, a link to a comprehensive tutorial on this is provided here.

    Once your installation is complete, click the image below should appear. Click on Python.mpkg.

    If you click on the Python 3 folder, you will get to the window displayed on the right hand side. Click on the IDLE and a window (on the left hand side of image) should appear.Type:
